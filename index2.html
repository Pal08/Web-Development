Module — 02

UI / UX Design

Figma

In this module I explored the principles and tools behind user interface and user experience design. Starting from design thinking and wireframing, to building interactive prototypes in Figma, I learned how to create interfaces that are both visually appealing and genuinely usable. The module covered the full design pipeline — research, ideation, prototyping, and testing.
Day by Day
Day 1
Introduction to Figma
  • Navigated the Figma interface and set up my first frame
  • Used Rectangle, Ellipse shape tools
  • Applied corner radius to create rounded UI elements
  • Added and styled text — font, colour, size, weight, and alignment
  • Drew custom shapes and icons using the Pen tool
Day 2
Red Bull Website
  • Designed a Red Bull branded website layout in Figma
  • Built a navbar with logo, nav links (Energy Drinks, Gallery, Our Story, Contact)
  • Added icon buttons — Search, Wishlist, and Cart
  • Created a bold blue gradient hero background with vertical stripe texture
  • Structured the Red Bull Sugarfree product page layout
Day 3
Red Bull — Product Page Design
  • Designed a full Red Bull Sugarfree product landing page
  • Built a navbar with logo, nav links, and icon buttons (Search, Wishlist, Cart)
  • Created a hero section with product image, title, description, price and Shop Now CTA
  • Added a product variant sidebar — Sugarfree, Energy Drink, Zero with ratings and price
  • Included social media links and a "Ready to take off?" footer section
Day 4
Red Bull Website
  • Designed 3 Red Bull product pages — Sugarfree, Energy Drink, and Zero
  • Each screen had a hero product image, title, description, and $2.99 price tag
  • Built a consistent navbar with logo, nav links, and Search, Wishlist, Cart icons
  • Added a product variant sidebar with star ratings and add to cart button
  • Kept a unified blue gradient background and layout across all three screens
Day 5
Glassmorphism in Figma
  • Created a glassmorphism card using dark frosted background and corner radius
  • Applied radial gradients to circles for glowing blob effects
  • Used red, purple, and blue colour blobs to create depth behind the glass
  • Experimented with opacity, blur, and layering to achieve the frosted look
  • Practiced visual composition by overlapping shapes on a dark canvas

My Reflection

Before this module, I thought design was mostly about making things look pretty. This module completely changed that mindset — good design is about solving problems for real users.

Learning Figma was exciting. The auto-layout feature especially felt like a superpower once I understood it — components that resize and adapt automatically saved so much time. Building a proper design system with consistent colors, type scales, and reusable components made me understand why big companies invest so heavily in this.

The most important lesson was that design is iterative. My first wireframes looked nothing like the final screens, and that's okay. Every round of feedback made the design sharper. I now approach every web project with a design-first mindset before writing a single line of code.
