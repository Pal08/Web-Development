Module — 01

Web Development

HTML · CSS · JavaScript · GitHub

In this module I learned the core foundations of building websites — from structuring pages with HTML, styling with CSS, to adding interactivity with JavaScript. Each day of the FabLab session focused on a different concept, progressively building toward a complete To-Do List project deployed live on GitHub Pages.
Day by Day
Day 1
Basic HTML Tags

Introduction to the building blocks of every webpage — writing your very first HTML document.

  • Set up VS Code & created first .html file
  • Used tags: p, b, i, u, br, hr
  • Text effects with marquee, sub, sup
  • Understood HTML document structure & nesting
  • Practiced heading tags h1–h6 & paragraphs
Day 2
Image, Anchor, Media & Table

Embedding rich content and linking pages — making HTML truly interactive and visual.

  • Inserted images with <img> & attributes
  • Created hyperlinks using <a href>
  • Embedded video & audio with <video>, <audio>
  • Built structured data tables — rows, cols, colspan
  • Added table headers, borders & captions
Day 3
HTML Forms

Collecting user input — the gateway to dynamic and interactive web applications.

  • Built forms with <form>, action, method
  • Input types: text, email, password, number, date
  • Used radio buttons, checkboxes & dropdowns
  • Added textarea, submit & reset buttons
  • Applied label, placeholder & required
Day 4
Header, Nav, Footer & CSS Selectors

Structuring pages semantically and targeting elements with the full power of CSS.

  • Semantic tags: header, nav, footer, main
  • CSS selectors: element, class (.), ID (#)
  • Pseudo-classes: :hover, :first-child, :nth-child
  • Attribute selectors & combinator selectors
  • Styled a full-page layout with nav & footer
Day 5
Box Model & Flexbox

Mastering layout — understanding spacing and building flexible, responsive arrangements.

  • Box Model: margin, border, padding, content
  • box-sizing: border-box explained
  • Flexbox: display:flex, flex-direction
  • justify-content, align-items, gap
  • Built a responsive card layout using Flexbox
Day 6
JavaScript Basics & All Operations

Bringing pages to life — writing logic, manipulating the DOM, and handling all core JS operations.

  • Variables: var, let, const & data types
  • Arithmetic, comparison & logical operators
  • Conditionals: if/else, switch
  • Loops: for, while, do-while
  • Functions, DOM selection & event listeners
Day 7
Final Project
To-Do List Project

The culmination of everything — combining HTML, CSS, and JavaScript into one complete, deployed web application.

  • Built a full To-Do List app from scratch
  • Add, delete & mark tasks as complete
  • Used localStorage to persist tasks on refresh
  • Applied HTML + CSS + JS together end-to-end
  • Pushed code to GitHub repository
  • Deployed live on GitHub Pages

My Reflection

This was my first real deep dive into web development. I had seen HTML before but never built anything complete from scratch. Working through this module day by day taught me how all three core technologies — HTML, CSS, and JavaScript — fit together to create a real working website.

The biggest challenge was understanding the Box Model and Flexbox. Getting elements to space and align properly took a lot of trial and error, but once it clicked it felt like a superpower — I could finally control the layout of any page with confidence.

Learning JavaScript on Day 6 was a turning point. Understanding variables, loops, functions, and DOM manipulation made me realize how much is possible with just vanilla JS — no libraries needed.
